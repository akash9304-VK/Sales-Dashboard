# Power BI Restaurant Sales Dashboard

## Overview
This project analyzes restaurant sales performance using Power BI.  
The dashboard provides insights into revenue, profit, orders, and average order value across multiple restaurants and cities.

## Key Metrics
- Net Profit
- Total Orders
- Average Order Value (AOV)

## Features
- Interactive filters for Month, City, and Restaurant
- Revenue analysis by city using map visualization
- Restaurant performance comparison using bubble chart
- Navigation pages for Summary, City, Restaurant, and Month

## Tools Used
- Power BI
- DAX
- Data Modeling
- Data Visualization
